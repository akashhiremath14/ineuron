Introduction
  A self-learning real-time interactive chatbot built on Skype-real-time-dataset of ineuron.

Objective
  * Easily Integrable
  *Easily Accessible to user
  *Self Learning
      *Understand Doubts
      *Find the more similar case
      *Manual intervention
      *Conversation Analysis / Feedback
      *Update Case Base
      
Requirements
* Heroku
* DialogFlow
* Yake
